**`upsertCharacter(projectId: string, character: Partial<CharacterRow> & { name: string })`:**
General-purpose character upsert for the story bible page (CHAR-01, CHAR-02).
1. Authenticate user, verify project ownership.
2. If character has `id`: update existing row.
3. If no `id`: insert new row with `source: 'manual'`.
4. `revalidatePath` for story bible page.
5. Return `{ success: true, id: string }` or `{ error: string }`.

**`deleteCharacter(characterId: string)`:**
1. Authenticate, verify ownership via project FK.
2. Delete from characters table.
3. Revalidate.

**`upsertLocation(projectId: string, location: Partial<LocationRow> & { name: string })`:**
Same pattern as upsertCharacter for locations.

**`deleteLocation(locationId: string)`:**
Same pattern as deleteCharacter.

**`upsertWorldFact(projectId: string, fact: { category: string; fact: string; id?: string })`:**
Insert or update world fact. Validate category is one of 'timeline' | 'rule' | 'lore' | 'relationship'.

**`deleteWorldFact(factId: string)`:**
Delete world fact by ID.
