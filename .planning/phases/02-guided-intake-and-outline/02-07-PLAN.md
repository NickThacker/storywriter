**Dialog content — three tabs or sections:**

1. **Full Regenerate** — "Start Fresh"
   - Description: "Regenerates the entire outline from your original intake data"
   - Single button: "Regenerate Outline"
   - On click: calls `onRegenerate('full')`

2. **Regenerate with Direction** — "Guided Regenerate"
   - Description: "Regenerate with your feedback. The AI uses your original data plus your direction."
   - Textarea for direction/feedback (placeholder: "Make act 2 more tense", "Add a subplot about...")
   - Button: "Regenerate with Direction" (disabled when textarea empty)
   - On click: calls `onRegenerate('directed', directionText)`

3. **Per-Chapter Regenerate** — "Redo This Chapter"
   - Only shown when `selectedChapterIndex` is provided
   - Description: "Regenerate just chapter {N} without affecting the rest"
   - Optional textarea for chapter-specific direction
   - Button: "Regenerate Chapter {N}"
   - On click: calls `onRegenerate('chapter', directionText)`

The parent OutlinePanel handles the actual generation by calling `startStream` from `useOutlineStream`. For per-chapter regeneration, the parent calls a separate endpoint or modifies the prompt to regenerate just that chapter — this is orchestrated in the parent, not in this dialog.

Include a warning: "This will replace your current outline. Your previous version will be saved." (because saveOutline snapshots to previous_chapters).

**`seedStoryBibleFromOutline(projectId: string, outlineData: GeneratedOutline)`:**

1. Authenticate user, verify project ownership.
2. **Characters:** For each character in `outlineData.characters`:
   - Check if a character with the same name already exists for this project (handles merge with manually created characters per research Open Question 3).
   - If exists AND source is 'manual': update only null fields (don't overwrite user's manual edits).
   - If exists AND source is 'ai': full update with new AI data.
   - If not exists: insert with `source: 'ai'`.
   - Fields from outline: name, role, one_line, arc. Other fields (appearance, backstory, personality, voice, motivations) are null — they can be filled in on the story bible page.
3. **Locations:** For each location in `outlineData.locations`:
   - Delete existing AI-generated locations for project, insert fresh.
   - Fields: name, description.
4. Return `{ success: true }` or `{ error: string }`.

Import `GeneratedOutline` from `src/lib/outline/schema.ts` (Plan 05).
