**Create `src/lib/billing/token-interceptor.ts`:**

A `TransformStream<Uint8Array, Uint8Array>` factory that:
1. Passes every chunk through immediately (never blocks the stream)
2. Accumulates SSE text in a buffer to parse line-by-line
3. Looks for the final chunk containing `usage.total_tokens`
4. When usage is found, calls the provided `onUsage` callback
5. In the `flush` handler, processes any remaining buffered text

```typescript
interface UsageData {
  prompt_tokens: number
  completion_tokens: number
  total_tokens: number
}

export function createTokenInterceptStream(
  onUsage: (usage: UsageData) => void
): TransformStream<Uint8Array, Uint8Array> {
  const decoder = new TextDecoder()
  let buffer = ''

  return new TransformStream({
    transform(chunk, controller) {
      controller.enqueue(chunk) // ALWAYS pass through first
      buffer += decoder.decode(chunk, { stream: true })
      const lines = buffer.split('\n')
      buffer = lines.pop() ?? '' // keep incomplete line in buffer

      for (const line of lines) {
        if (!line.startsWith('data: ')) continue
        const data = line.slice(6).trim()
        if (data === '[DONE]') continue
        try {
          const parsed = JSON.parse(data)
          if (parsed.usage?.total_tokens) {
            onUsage(parsed.usage)
          }
        } catch { /* not JSON, skip */ }
      }
    },
    flush() {
      // Process any remaining buffer
      if (buffer.startsWith('data: ')) {
        const data = buffer.slice(6).trim()
        if (data !== '[DONE]') {
          try {
            const parsed = JSON.parse(data)
            if (parsed.usage?.total_tokens) {
              // This edge case: usage in very last unflushed chunk
            }
          } catch { /* ignore */ }
        }
      }
    },
  })
}
```

**Create `src/lib/billing/budget-check.ts`:**

Two functions:

1. `checkTokenBudget(userId: string)`: Server-side function called before generation.
   - Create Supabase server client
   - Fetch `user_settings` for the user: `openrouter_api_key`, `subscription_tier`, `token_budget_remaining`, `token_budget_total`, `credit_pack_tokens`
   - If `openrouter_api_key` is set (truthy), return `{ allowed: true, isByok: true }` -- BYOK bypasses billing entirely
   - If `subscription_tier === 'none'` and no API key, return `{ allowed: false, isByok: false, reason: 'no_subscription' }`
   - Calculate effective remaining = `token_budget_remaining + credit_pack_tokens`
   - If effective remaining <= 0, return `{ allowed: false, isByok: false, reason: 'budget_exhausted' }`
   - Calculate usage percent: `(budgetTotal - token_budget_remaining) / budgetTotal` (ignore credit_pack_tokens in percentage)
   - If usage percent >= 0.8, return `{ allowed: true, isByok: false, warningThreshold: 'near_limit', budgetRemaining: token_budget_remaining, budgetTotal }`
   - Otherwise return `{ allowed: true, isByok: false, warningThreshold: null }`

2. `deductTokens(userId: string, totalTokens: number)`: Called after generation completes (fire-and-forget from the interceptor).
   - Fetch current `token_budget_remaining` and `credit_pack_tokens`
   - Deduct from `token_budget_remaining` first. If that goes to zero, deduct overflow from `credit_pack_tokens`.
   - Update `user_settings` with new values
   - Use `Math.max(0, ...)` to prevent negative values

3. `recordTokenUsage(data: { userId, projectId, chapterNumber, promptTokens, completionTokens, totalTokens })`: Insert a row into `token_usage` table.
   - Use Supabase service role client (same pattern as webhook handler) since this runs in route handler context and needs to bypass RLS for inserts.

Import `createClient` from `@supabase/supabase-js` for service role operations. Import `createClient` from `@/lib/supabase/server` for user-scoped reads.

     // BYOK: passthrough, no tracking
     return new Response(responseBody, { ...sseHeaders })
   }

   // Hosted tier: intercept tokens
   const interceptedStream = responseBody.pipeThrough(
     createTokenInterceptStream((usage) => {
       // Fire-and-forget: don't await, don't block the stream
       recordTokenUsage({
         userId: user.id,
         projectId,
         chapterNumber,
         promptTokens: usage.prompt_tokens,
         completionTokens: usage.completion_tokens,
         totalTokens: usage.total_tokens,
       }).catch(err => console.error('Token recording error:', err))

         .catch(err => console.error('Token deduction error:', err))
     })

     'Content-Type': 'text/event-stream; charset=utf-8',
     'Cache-Control': 'no-cache, no-transform',
     Connection: 'keep-alive',
     'Content-Encoding': 'none',
   }

     headers['X-Budget-Warning'] = 'near_limit'
