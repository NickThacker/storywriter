**Derive chapter list items:**
```typescript
function deriveChapterList(): ChapterListItem[] {
  return outlineChapters.map((ch) => {
    const checkpoint = checkpointMap.get(ch.number)
    const wc = checkpoint?.chapter_text
      ? checkpoint.chapter_text.trim().split(/\s+/).filter(Boolean).length
      : 0
    let status: ChapterListItem['status'] = 'pending'
    if (generatingChapter === ch.number) {
      status = 'generating'
    } else if (checkpoint && checkpoint.chapter_text) {
      status = 'checkpoint'  // Phase 4 adds 'approved'
    }
    return { number: ch.number, title: ch.title, status, wordCount: wc, hasText: !!checkpoint?.chapter_text }
  })
}
```

**Generation flow:**
1. User clicks Generate on a chapter → set `generatingChapter = chapterNumber`, call `startStream(projectId, chapterNumber)`
2. While streaming → right panel shows `ChapterStreamingView`
3. On stream complete (isStreaming goes false, streamedText is non-empty):
   a. Call `/api/generate/compress-chapter` POST with `{ projectId, chapterNumber, chapterTitle, chapterText: streamedText }` — this saves the checkpoint + updates memory
   b. On success: update `checkpointMap` with the new text, call `updateProjectWordCount(projectId)` to sync totals, clear `generatingChapter`
   c. On error: show toast with error, keep the streamed text visible

4. Use `useEffect` watching `[isStreaming, streamedText]` to detect stream completion and trigger the compression call.

**Right panel logic:**
- If `generatingChapter !== null` (streaming or just completed): show `ChapterStreamingView`
- Else if selected chapter has text in `checkpointMap`: show `ChapterEditor` with `initialContent` = checkpoint text
- Else: show an empty state with "Select a chapter and click Generate to start writing"

**Editor save handler:**
```typescript
const handleEditorSave = async (text: string) => {
  const chapterNumber = outlineChapters[selectedIndex].number
  await saveChapterProse(projectId, chapterNumber, text)
  // Update local checkpoint map
  setCheckpointMap(prev => {
    const next = new Map(prev)
    const existing = next.get(chapterNumber)
    if (existing) {
      next.set(chapterNumber, { ...existing, chapter_text: text })
    }
    return next
  })
}
```

**Rewrite flow:**
- ChapterList or editor toolbar has a "Rewrite" button (visible for chapters with existing text)
- Opens `RewriteDialog` → user enters adjustments → calls `startStream(projectId, chapterNumber, adjustments)`
