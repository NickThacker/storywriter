Create a `'use client'` component that provides a rich text editing experience for chapter prose.

**Props interface:**
```typescript
interface ChapterEditorProps {
  initialContent: string   // Plain text from chapter_text column
  onSave: (text: string) => Promise<void>  // Called on debounced save
  readOnly?: boolean       // Disable editing (during streaming)
}
```

**Custom Extensions:**

1. **SceneBreak** — A block-level node that renders as a `<hr class="scene-break">` element. Visually renders as centered `***` via CSS.
   ```typescript
   const SceneBreak = Node.create({
     name: 'sceneBreak',
     group: 'block',
     atom: true,  // Cannot contain content
     parseHTML() { return [{ tag: 'hr.scene-break' }] },
     renderHTML() { return ['hr', { class: 'scene-break' }] },
     addCommands() {
       return {
         insertSceneBreak: () => ({ commands }) =>
           commands.insertContent({ type: this.name }),
       }
     },
   })
   ```

2. **AuthorNote** — A block-level node that contains inline content, rendered as an `<aside data-author-note>` element. Styled with a yellow/amber background, italic text, and a label.
   ```typescript
   const AuthorNote = Node.create({
     name: 'authorNote',
     group: 'block',
     content: 'inline*',
     parseHTML() { return [{ tag: 'aside[data-author-note]' }] },
     renderHTML() { return ['aside', { 'data-author-note': '', class: 'author-note' }, 0] },
     addCommands() {
       return {
         insertAuthorNote: () => ({ commands }) =>
           commands.insertContent({ type: this.name, content: [{ type: 'text', text: '' }] }),
       }
     },
   })
   ```

**Editor setup:**
```typescript
const editor = useEditor({
  extensions: [StarterKit, SceneBreak, AuthorNote],
  content: initialContent,  // Tiptap wraps plain text in <p> tags automatically
  immediatelyRender: false,  // CRITICAL: prevents Next.js SSR hydration error
  editable: !readOnly,
  onUpdate({ editor }) {
    const text = editor.getText()  // Plain text only — NOT getHTML()
    debouncedSave(text)
  },
})
```

**Toolbar:** Create a simple toolbar above the editor with buttons for:
- Bold (B), Italic (I), Heading 2 (H2), Heading 3 (H3)
- Scene Break (horizontal rule icon) — calls `editor.chain().focus().insertSceneBreak().run()`
- Author Note (message-square icon) — calls `editor.chain().focus().insertAuthorNote().run()`

Use `lucide-react` icons: `Bold, Italic, Heading2, Heading3, Minus, MessageSquare`.
Style toolbar buttons with active state highlighting (check `editor.isActive('bold')` etc.).

**Auto-save:**
```typescript
const debouncedSave = useDebouncedCallback(async (text: string) => {
  await onSave(text)
}, 600)
```

**Styling:** Use Tailwind `prose prose-sm max-w-none` on the EditorContent wrapper. Add custom CSS for:
- `.scene-break`: `border-none, text-align: center, margin: 2rem 0` with `***` rendered via `::after` pseudo-element or as a styled hr
- `.author-note`: `bg-amber-50 dark:bg-amber-950/20, border-l-2 border-amber-400, px-3 py-2, italic, text-sm text-amber-800 dark:text-amber-200`

Add the styling inline using a `<style>` tag in the component or via Tailwind `@apply` in a scoped style block. Prefer inline styles object or a Tailwind-based approach using the `className` prop on EditorContent and CSS-in-JS for the custom node styles.

**Word count display:** Show a small word count badge below the editor (bottom-right), computed from `editor.getText()`.

**Export:** `export function ChapterEditor({ initialContent, onSave, readOnly }: ChapterEditorProps)`

**CRITICAL PITFALLS to avoid:**
- Do NOT use `editor.getHTML()` for saving — use `editor.getText()` (plain text for DB storage)
- Do NOT omit `immediatelyRender: false` — will cause hydration errors
- Do NOT use `StarterKit.configure({ history: false })` — Tiptap 3 renamed this to `undoRedo`
